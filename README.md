# AI-TRIZ 创新引擎 (AI-TRIZ Innovation Engine)

[![English](https://img.shields.io/badge/Language-English-blue)](#english) [![中文](https://img.shields.io/badge/语言-中文-red)](#中文)

## English

An innovative web application that combines Artificial Intelligence (AI) with the Theory of Inventive Problem Solving (TRIZ). This engine is designed to help engineers, researchers, and innovators systematically deconstruct problems, derive creative solutions, and gather cross-domain inspiration.

### Features
- **Problem Deconstruction**: Systematically break down complex engineering or business problems into manageable components and contradictions.
- **Solution Derivation**: Utilize TRIZ principles powered by AI to generate actionable and innovative solutions.
- **Cross-Domain Inspiration**: A dedicated inspiration board to discover analogies and solutions from unrelated industries.
- **System Settings**: Customizable AI parameters and user preferences.

### Tech Stack
- **Framework**: Vue 3
- **Routing**: Vue Router
- **UI Components**: Element Plus (Recommended)
- **Data Visualization**: ECharts (Recommended)

### Getting Started

#### Prerequisites
- Node.js (v16+)
- npm or yarn

#### Installation

```bash
# Clone the repository
# git clone https://github.com/yourusername/ai-triz-engine.git

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## 中文

一个将人工智能（AI）与发明问题解决理论（TRIZ）相结合的创新型 Web 应用。本引擎旨在帮助工程师、研究人员和创新者系统化地解构问题、推导创新方案，并获取跨界的灵感。

### 核心功能
- **问题解构**：将复杂的工程或业务问题系统性地拆解为可管理的组件和技术矛盾。
- **创新方案推导**：利用 AI 驱动的 TRIZ 原理（如 40 个发明原理），生成具备落地可行性的创新方案。
- **跨界灵感看板**：专属的灵感探索区，帮助用户从不相关的行业中发现类比方案和跨界灵感。
- **系统设置**：支持自定义 AI 参数及用户偏好配置。

### 技术栈
- **前端框架**：Vue 3
- **路由管理**：Vue Router
- **UI 组件库**：Element Plus（首选）
- **数据可视化**：ECharts（首选）

### 快速开始

#### 环境要求
- Node.js (v16 及以上版本)
- npm 或 yarn

#### 安装与运行

```bash
# 克隆项目
# git clone https://github.com/yourusername/ai-triz-engine.git

# 安装依赖
npm install

# 启动本地开发服务器
npm run dev

# 构建生产环境代码
npm run build
```
